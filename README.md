siderealsystem
==============

Online Planetarium

**Stack:** Vite, TypeScript, Svelte 5.

## Development

```bash
pnpm install
pnpm dev
```

Build for production and preview the static output:

```bash
pnpm build
pnpm preview
```

Lint, typecheck (Svelte + TS), and unit tests:

```bash
pnpm lint
pnpm check
pnpm test
pnpm test:watch
```
